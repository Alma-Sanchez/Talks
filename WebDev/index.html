What is web development?


Web development is the process of developing any application for the WWW (World Wide Web)

Similarities to Application Development

  • Interface
  • Stores/Retrieves Data Somewhere
  • Buttons that lead to events
  • A server that hosts the website or a downloadable file (app store)

Components of Web Dev

  • Appearance
  • Behavior & User Interaction
  • Data Manipulation
  • Deployment
